# Toast

一个简洁的消息，短暂显示。

## Preview

<Canvas of={ToastStories.Default} />

## Usage

### 将 `Toaster` 组件添加到您的布局中

export const providerCode = `import { Toaster } from "@una/ui/toast";

export default function RootLayout({ children }) {
  return (
    <html lang="en">
      <head />
      <body>
        <main>{children}</main>
        <Toaster />
      </body>
    </html>
  );
}`;

<Source code={providerCode} language='tsx' dark />

### 使用 `toast` 函数从 `useToast` 钩子中

export const importCode = `import { useToast } from "@una/ui/toast";`

export const usageCode = `export const ToastDemo = () => {
  const { toast } = useToast();

return (

<Button
  onClick={() => {
    toast({
      title: 'Scheduled: Catch up',
      description: 'Friday, February 10, 2023 at 5:57 PM',
    })
  }}
>
  Show Toast
</Button>
); } `;

> 要同时显示多个 toast，可以更新 `use-toast.tsx` 中的 `TOAST_LIMIT`。
