# Badge

显示一个徽章或看起来像徽章的组件。

## Preview

<Canvas of={BadgeStories.Default} />

<Controls />

## Usage

export const importCode = `import { Badge } from "@una/ui/badge";`

export const usageCode = `<Badge variant="outline">徽章</Badge>`

<Source code={importCode} language='ts' dark />
<Source code={usageCode} language='tsx' dark />

## Helper

你可以使用 `badgeVariants` 助手来创建一个看起来像徽章的链接。

export const helperImport = `import { badgeVariants } from "@una/ui/badge";`

export const helperUsage = `<Link className={badgeVariants({ variant: "outline" })}>徽章</Link>`

<Source code={helperImport} language='ts' dark />
<Source code={helperUsage} language='tsx' dark />
