# Avatar

一个带有备用的图像元素，用于表示用户。

## Preview

<Canvas of={AvatarStories.Default} />

<Controls />

## Features

- 自动和手动控制图像渲染。
- 备份部分接受任何子元素。
- 可选延迟备份渲染以避免内容闪烁。
